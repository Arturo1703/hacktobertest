Lets learn a bit about machine learning using a very high-level library , called ml5 by Danniel Siffman.

In these brief tutorial we will be doing some examples on these library which is on top of tensorflow.js

This library uses p5.js and pre-trained models to abstract from the mathematics of machine learning. If you want to feel the pain of learning the math, stay tuned to SciData
When you feel free to do the first exercise go to:

https://github.com/dlgeraghty/hacktobertest/tree/master/learning-ml-5/image-classification

Make sure you read the readme file!!
